# Virtual Phone Demo

This project is a simple prototype of a virtual phone interface built with a single HTML file (`Phone.html`). It showcases a phone-style layout and basic UI elements for experimentation and learning.

## Getting Started

1. Open `Phone.html` in your browser.
2. Interact with the on-screen elements to explore the demo.
